# Gilroy Font Files

Đặt các file font Gilroy (.ttf) vào thư mục này:

## Required Files:
- `Gilroy-Light.ttf` (weight 300)
- `Gilroy-Regular.ttf` (weight 400)
- `Gilroy-Medium.ttf` (weight 500)
- `Gilroy-SemiBold.ttf` (weight 600)
- `Gilroy-Bold.ttf` (weight 700)

## Download Sources:
- [BeFont - Gilroy Font](https://befonts.com/gilroy-font-family.html)
- [DaFont - Gilroy](https://www.dafontfree.io/gilroy-font-free/)

## License:
Gilroy Light và ExtraBold miễn phí cho personal use. Nếu dùng commercial, cần mua license.

## Instructions:
1. Download font files từ nguồn trên
2. Extract file .zip
3. Copy các file .ttf vào thư mục này
4. Rebuild app: `npx expo start --clear`
